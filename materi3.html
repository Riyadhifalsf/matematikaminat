Perpotongan garis dan lingkaran

Suatu lingkaran dengan persamaan x2 + y2 + Ax + By + C = 0 dapat ditentukan apakah suatu garis h dengan persamaan y = mx + n tersebut tidak menyentuh, menyinggung, atau memotongnya dengan menggunakan prinsip diskriminan. Dari persamaan kuadrat dapat dilihat apakah garis tidak menyinggung/memotong, menyinggung atau memotong lingkaran. Garis h tidak memotong/menyinggung lingkaran, maka D < 0 Garis h menyinggung lingkaran, maka D = 0 Garis h memotong lingkaran, maka D > 0
